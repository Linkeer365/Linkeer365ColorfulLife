深夜的麦当劳-等差数列

{系列2}

“真不好意思这么晚的雨夜还让您奔波…”
“对不起, 实在是饿得受不了…”
“请原谅我…”

尽管雨一直保持着”初雪”的状态似乎对外卖叔叔影响会小一点,然而这一来不能justify我叫外卖的动机,二来要是回程途中雨势增大, 仍然是极其狼狈的.
饥饿感消除之后的我良心逐渐回归, 一种深切的羞愧感像蒸汽一样从脚趾间一刻不停地向上升,
这就是我的举措吗? 这还是以前那个我吗?
我的大脑开始疼起来了, 我不知道说了多少句”请原谅我”依旧不管用
但我显然是太过激动了, 回过头来外卖叔叔眼神直勾勾的盯着我,已经怔住了神, 他把滚烫的外卖塞在我的手里

滚烫的外卖是我全身上下唯一的热源, 我的心早已冷透了

“哪里呀小兄弟, 干我们这行的赚的就是这份工资呀…”
“可是天也这么晚了又下雨, 我是实在忍不住饿我…”
“叔叔也知道这份工作辛苦而又危险, 但是有什么办法呢? 人总要挣口饭吃才行…”

这番话使我的内心像开满了监狱门墙盛放的蔷薇花一样,据说上面滴下的泪水会让瞎子复明,
我把自己的伞戳进他的怀里, 我根本不敢看他的脸, 后面我几乎是嘶吼着对他说,
“快走吧! 伞是你的! 再不走雨一大这把小伞根本不顶用!”

外卖叔叔估计根本没有看过这个阵仗, 余光里我只看到他努力洗一洗鼻子, 然后放一张红纸在我外卖盒子里, 估计是优惠券什么的,
“有你这些话我心里别提有多好受了, 是我要感谢你还来不及呢…”
“叔叔以前也喜欢托尔斯泰, 这点好就好在显得生活没那么苦…”

于是我们稀里糊涂间互加了对方的微信好友, 我寻思着过一会问个平安,然后过三五个月转一点钱过去, 外卖员微信朋友多而驳杂, 到时候胡编个理由说忘记付款什么的就好
这样我的良心, 才终于得到了片刻的安定, 但仅仅是策划而已, 到了落地之时, 估计又有一番风波
我带着沉重稍解的心情来到宿舍, 热腾腾的晚餐并没有带来多少快意, 我只是机械性的把肚子填了个饱

一阵饱嗝后我收拾了餐盒, 剩下的仅仅是一张红纸和一张祝福语的标签贴,
标签贴上自然是客套的老话了,像是”永劫长明”什么的, 署名是”天地缝隙的羔羊”, 我不禁苦笑, 托尔斯泰害人呀, 这个场合下怎么会用这种署名呢?然而更苦于这恰如其分的所指.
正因为永劫不能长明, 所以天地缝隙间才会有羔羊吧.
那张红纸也并不是优惠券,上面歪七扭八地写着一些东西, 仔细一看是一个Java实现的等差数列, 估计是外卖叔叔的儿子写的然后不慎带出的吧…

叔叔,谢谢你,愿你今晚有个好梦.

在某一瞬间我深深地感觉到, 要是我能够不理会这一系列的蹩脚把戏, 那些闲事会不会晚一点才能找到我;
要是我能够迟钝到无以复加,是不是就能沉浸在对大叔的愧疚与祝福里安然度过一些日子, 说不定甚至是一辈子.

带着这样的想法, 我一脚踹进了学校的咖啡馆,一阵咳嗽声传来

“喝你妈呢!”
距离上次的见面仅仅过了十年, 那么他的容貌应该没有明显的变化,我拐进咖啡店的角落
一个怯生生的、穿着精致哥德风格衣裙的身影浮现在我的眼前, 蝴蝶结搁在一旁没有戴上, 这样我就无法很轻易地抓住他的衣领
不用问了, 这种并不刻意收束腰身, 却在走线细节上异常刻板的礼服, 已使我锁定了对象

“等!差!数!列!”

“看起来你应该刚吃饱, 要不要喝点什么”
我把我破门而入的话重复了一遍, 悲痛转化为的怒气总不那么好消除,
“看起来你火气很大, 不如来杯凉茶消消火”
欠揍的笑容以指数增长的高速, 像癌细胞一样弥散在他的脸上, 一路延伸到脖子

在讹诈了两杯冰茶和一支冰淇淋之后, 我的怒火和他的笑意都渐渐散去,
“听说甜品是一种能够暴力带走忧伤的物质, 今天终于得到了印证…”
“不过话说回来, 如果破绽太多的话, 其实反倒更应该多加防范的不是吗? 那种级别的破绽,几乎都跟提示没什么区别了…”

的确, 种种破绽现在想来低级得可笑,
且不说这种浅显而低幼的符号化风格, 一看就出自他的手笔,类似于”天地缝隙的羔羊”什么的, 又比如等差数列和Java等等,
从取餐之前没有收到任何的电话, 售货员对收餐者丝毫不加以确认, 深夜外卖竟然能够做到烫手的地步,外卖人员竟然会和不相识的顾客从容聊起托尔斯泰等种种疑点来讲, 我早就应该意识到了,
然而沉浸在悲伤当中的我却没有丝毫的反应, 直到发现Java的关键提示才如梦方醒.
想到这一节, 自己今夜一切的忏悔与感动自然都变成了一场笑话, 我又产生了灭口的想法.

“别这么生气嘛, 你只是太久没有收到这样的刺激了, 像是巴弗洛夫如果长时间不训狗, 狗的反射也会退化…”
他嘲讽般的笑意, 预先缓存了一份在眉宇间, 以便日后调用, 而他的神情却逐渐严肃起来.
他用力吸一吸鼻子, 试图打破这个到目前为止只有他发言的尴尬局面,
“这是我设置的小测试, 显然对你来说太过轻松了”

虽然依旧没有想要开口的心思和必要,但看在他年纪小的份上,我也就勉为其难说上几句好了
毕竟长兄如父,在这一点上他没有任何话语权可言,

“唉, 这十年没能把你塑造成社会的栋梁之才, 反而让你走上邪路成了变态!这一切,实在是爸爸的失职…”
“十年前的你仅仅是蠢蠢欲动而已, 没想到…唉…”
“看着你一步步堕落的过程, 不得不令我感到时光飞逝,岁月如歌…”

说实在的, 他身着的那套哥特萝莉风的服饰在精致之余其实并不出彩, 因为这个款式至少十年前就已经有了, 并且彼时他就进行着有限度的尝试, 现在技术更加纯熟也是意料之中.
加之以他年龄与容貌的极缓慢增长, 此类装束便产生了buff般的超永久续行力, 这么说吧,至少在我老死以前, 他穿着这一身都不会显出突兀感.

“等差数列”是”观布子”这个特殊人类族群的一员, 这个族群正是所谓的”神之信使”, 一来乃是他们能够随着年龄加载很多既有的或者自行设计的神通能力模块, 扩展了人类的极限;二来则是因为他们本性的无私与无畏, 这估计也与对神通能力模块加载的能力有关, 观布子人并非不死族, 然而他们15岁时能够即时加载的一个history模块, 使得他们能够访问前些辈子的所作所为, 这种”连续性历史记忆”才是那颗最大的定心丸, “又是一条好汉”对观布子族而言绝非戏言, 而是久经证实、不言而喻的真理.

然而最为生动且最令我印象深刻的, 则是他们特殊的年龄增长机制,事实上这也是他”等差数列”诨名的由来,进一步的实情乃是这一机制的家喻户晓,近乎所有经历过高等教育的学子们对”等差数列”的第一印象,一定也来自这个神奇的人类族群.

“已知一个观布子人从10岁开始具备时间抗性, 这种抗性可以近似为: 每增长1岁, 需要耗费与之前年龄相同的年份数; 请问1. 100年后这个观布子人的岁数是多少? 2. 观布子人活到100岁, 需要耗费多少年?”
这是等差数列开课前的一道引入例题, 精要地概括了观布子人的年龄增长机制, 同时也顺便考察普通的等差数列求和, 加之以此后层出不穷的种种习题, “观布子科学家们加载了某一特殊的时间抗性模块, 他们的年龄如下图所示, 请问他们近半数人100岁一共至少需要多少年”等等考察差比数列的, “中国70岁的观布子老爷爷向孙子辈们讲述着自己亲身经历的西汉七国之乱, 并声称担任过路中大夫的幕僚, 请问老爷爷说谎了吗? 为什么?{已知: 幕僚的年龄下限为25岁}”等等考察递推分析的…尽管观布子人群因此受到许多来自高中生的不白的诅咒, 这些诅咒却也反映出凡人们对观布子族群的深切的爱戴、憧憬与感激.

作为和我同一年出生的观布子人, “等差数列”与我分别时只有10岁, 于是这十年他恰好增长了1岁, 于是我就莫名变成了大他9岁的兄长.
冰淇淋的形状彻底无法稳定住了, 我把冰淇淋搅拌在冰茶里, 渐渐浑浊的液面里的偶发的丁达尔效应才使我注意到太阳已经升起, 回过头看见他在拍打着外卖制服,
“不出去晒晒?”
“昨天根本没雨, 只有你哭得跟个神经病一样, 我还以为早被你识破了,反倒耍我来着”
“既然没有雨, 地板上为什么会有这么一大摊积水?”

他突然沉默下去, 随后手抱后脑、伸着懒腰, “这不重要, 话说你最近都在忙些什么呀~”
我知道接下来可能会发生一段意义不明的闲谈, 我轻轻摆放好心中的蔷薇.

三个宏大的目标

发表于 2020-01-12 20:39:20 | 更新于: 2020-02-27 19:30:19

这封信主要包括什么

这封来信，主要想讲的是未来我想要参与达成的三个宏大的目标。这些目标，从所涉领域来看，有的仅仅与计算机科学相关，有的则是要依赖细粒度的计算机科学教育的普及才能达成；从运营成本及成果转化来看，有的能够由我及一小群志同道合者共同实现，有的则需要几代人的努力。

写这封信的发心是什么

当然，我不否认自己的私心，想要通过深切的自白，来弥补我绩点薄弱、计算机基础较差这一点，来向您申请一封推荐信。

但实际上，纵使我已经获得了出国留学的offer而不需要您的推荐信，或者我已下定决心不打算出国留学两种情况下，这些目标也不会改变的。

因为我还是一个年轻人，年轻人总是有些想着改变世界的理想与热血的。

哪三个目标

目标1：（针对未受过计算机专业教育的人群）

建立知名度广、话题度高的一款软件推荐网站（平台）

背景

我国计算机的普及率不高，造成了人们常常在“使用软件解决问题”上缺乏足够的认识，对“何种软件能够帮他们办成何种工作”没有足够强烈的主动探究意识。但是客观上，网络早已使得这一系列的工具软件可以被获取。举例如：

Software name Usage Home website
Listary File Launcher Listary
Filegee Files autosync and Continuous Integration Filegee
Onedrive Files autosync and Continuous Integration Onedrive
Directory Opus Enhanced Explorer Directory Opus
Total Uninstaller Files' Garbage Collection & Tracer Total Uninstaller

一系列成熟、授信、易部署的工具软件，能够充分简化纷繁重复的工作流程。

分析

我个人觉得，之所以存在许多深陷复杂工作流的人，仅仅只是因为他们没有获取此类信息的渠道。而之所以缺少渠道，主要原因是相比raddit等国外网站上的讨论以外，在中国这样的讨论较为稀少。尽管国内已存在一部分软件推荐平台，如少数派、善用佳软等，但影响力仍十分有限。

实现形式

因此，我们需要建立知名度广、话题度高的一款软件推荐网站，支付宝“扫一扫”的深入群众，证明人们是很容易接受生活方式上的深刻改变的，而支付宝的普及，可以作为一个参考目标。

目标2： （针对贫困山区孩子）

“图书中介”——“试验模式”的初步探索

背景

山区的贫困现状受多种因素影响，与外界的信息不对称我认为是关键的因素之一，而摆脱贫困，立竿见影的办法便是发展教育，这不仅仅需要师资力量与硬件设施的物质保障，科学普及与人文熏陶等精神培养更不能落下。我觉得我们国家在后者方面有所欠缺。

实现形式

我想通过“给贫困山区捐赠影印图书”的方式，来扩宽山区孩子们信息渠道，进而实现对后者的补益与支持，具体操作为：

由政府设计一个“图书中介”的部门（也可以是公益等第三方团体），电子信息资料的持有者负责提供电子材料，第三方负责给将其打印并捐助给山区学生，山区学生接收到书后，可选择地与捐赠人或第三方平台建立联系。

分析

我的想法基于以下两点：

  1. 在中国，书籍材料价格较低，而影印版书籍，价格则更为低廉。

贫困家庭的孩子不是因为买不起书，而是缺少得知这些资源的途径。贫困家庭的投资不是因为不愿意购置图书，而是样品稀缺无法获取。

  1. 在中国，物流的安全性几乎不受恶劣地理位置条件所限制。

巨大体量的交通运输业务，促使物流面向全国辐射，通往贫困山区的路远远不止一条。

这件事是需要三方长期配合共同努力的，我认为尽管中国能在50年内形成一种基本的“互帮互助”的“试验模式”都非常困难，但是这却是一件功在当代、利在千秋的一件事。

目标3：（针对计算机专业毕业生就业引导）

降温与回暖

背景1

中国的计算机行业，是充分借助了物联网、互联网等数个“东风”发展起来的，而不是靠着重大的科研成果、技术突破起家。尽管如此，单就体量上来看，依旧是迅速扩张、亢奋拔节的态势。这反映出中国计算机行业重度依赖互联网、依赖市场、依赖“风口”的，同时也掩盖了中国计算机科学及计算机教育起步晚、先天不足、技术储备不够雄厚的事实。

背景2

人才同样具有逐利性。风口上，一时间大量人才涌入互联网、深度学习、云计算等若干个“热门”“高薪”领域，造成相关岗位严重饱和；同时，许多较为冷门的领域则因为社会关注度不够及缺乏稳定投资等因素，长时间坐“冷板凳”而无人光顾。

长久而言，这将影响到国家长期的信息战略部署与储备，是十分危险的一件事。

背景3

企业的成功与高效，核心取决于业务模型的精巧性与健壮度，由业务模型衍生演化出来的一套业务逻辑，也自然是包含了充分适应客观市场、具备良好性能与各类高可用特性的客观需要，因此，大量的人才被吸纳进入产业链里，完成对这一套逻辑的实现与验证。这是企业的发展要义，充分精密充分安全，但是它的本质始终是业务导向的，即不可能对百年内没有明显回报的项目进行足额投资与充分反哺，企业主是不可能自发去修铁路的。

分析1

就国家层面而言，计算机基础层面的研究人才不可或缺，而在现如今如火如荼的市场化竞争下，大量人才被互联网巨头所吸纳，这潜在地造成了基础层面的人才流失，“坐冷板凳”的人才纷纷“背叛师门”，转投互联网，拥抱百万年薪，我认为这种现象是值得担忧的。

分析2

我认为，计算机学生就业有这几个方面是过热的：

  1. 互联网+

互联网没有深奥的技术需求，新的业务模型无法推动计算机科学的变革。

  1. 深度学习、人工智能方面研究

这些方面需要较深的数理逻辑知识储备，然而由于企业借东风，开设大量低端岗位抢占市场，大量不了解基本原理的人群涌入，泡沫现象严重。

  1. 云计算

充分发达的信息安全技术保障，是中国各大云平台当前普遍缺乏的一环，大量云平台根本没有相匹配的安全保障措施，这个问题无法解决，中国数十年后的云平台积累的安全漏洞将可想而知。

我认为，这几个方面是过冷的：

  1. 程序语言理论研究（Programming Language）

包括形式验证（Formal Proof）与静态分析（Static Analysis）等方面

典型“冷板凳”，在国内一直没有得到重视

  1. 密码论与信息安全研究（Information Security）

包括硬件安全（Hardware Security）等

  1. 芯片与半导体设计（IC Design）

实现形式

  1. 国家（教育部）行政手段制约市场化人才富集，回归理性：
  2. 产业战略转型，减少人才在互联网、深度学习、云计算上的富集，引导人才在计算机基础层面上做更深入的研究

寒冬中我们能做些什么吗？

当然，我本身学历不高，三个目标也仅仅是一己之见，有着诸多偏颇与片面性。但这确实反映了我对中国当下问题的一点思考，而今寒冬已至, 从制造业到互联网、金融都举步维艰，经济将怎么转型？市场何时回暖？这些都是不折不扣的“切肤”问题。从我的角度，我觉得要靠着技术成果的合理分享、教育的社会动员、深入计算机基础研究三方面合力，才有可能度过隆冬。尽管我对自己没有自信，但是我将这些见解发送给您，相信您有资源有能力纾解这些难题，也相信我们能够一起努力，尽可能少颠簸地平稳过冬。
